# C# and .NET Framework Projects

This repository contains the core projects completed during my C# development course.

## Projects:

### 1. Car Insurance Quote App
An MVC web application that calculates insurance quotes based on user input and specific business logic (age, car make/model, DUI history). 
* **Key Features**: Automated quote calculation, Admin dashboard for staff, and data persistence using Entity Framework.

### 2. Server Time App
A Razor Pages application built to demonstrate real-time server-side data rendering.
* **Key Features**: Utilizes `DateTime.Now` to display dynamic server time to the user.

### 3. Student Database (Code-First)
A console application demonstrating the Entity Framework Code-First workflow.
* **Key Features**: Automatically generates a database schema from C# classes and handles data entry for student records.
